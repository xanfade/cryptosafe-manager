
# CryptoSafe Manager

CryptoSafe Manager — учебный/проектный менеджер секретов (паролей) с упором на безопасность, расширяемость и постепенное развитие по спринтам.
Цель — построить модульную архитектуру, где криптография, база данных и GUI легко заменяются/расширяются без больших рефакторингов.

---

## Видение проекта

Приложение предоставляет:
- Защищённое хранилище записей (логин/пароль/URL/заметки/теги)
- Шифрование конфиденциальных полей перед сохранением в SQLite
- Журнал аудита действий (добавление/изменение/удаление/вход/выход)
- Настройки безопасности (таймер буфера обмена, авто-блокировка, политика доступа)
- Импорт/экспорт и резервное копирование (к финальным спринтам)
- Подготовку к финальной сборке (CI, упаковка)
